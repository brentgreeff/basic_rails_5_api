# Group events

Basic Rails API for creating, updating and deleting Events.

## Install

I am using rvm, but its not required.

      bundle install

      rake db:create

      rake db:migrate



## Approach

Please run `guard` when changing code.

press [enter] to run all tests.

The tests should be self documenting.
